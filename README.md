A fun and interactive memory game built using HTML, CSS, and JavaScript.
Test your memory by repeating an increasingly complex sequence of colors!

🧠 How to Play
Press any key (or click Start) to begin the game.
The game will highlight a color.
Repeat the sequence by clicking the buttons.
Each round, a new color is added to the sequence.
If you click the wrong color, the game ends.

🎯 Features
Dynamic color sequence generation
Sound effects for each button
Level progression system
Game over animation and restart
Responsive design
🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling and animations
JavaScript (ES6) – Game logic and interactivity
